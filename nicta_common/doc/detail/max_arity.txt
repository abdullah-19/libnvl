NICTA_MAX_ARITY:  a define for limiting number of iterations of the Boost.PP library
when used to generate variable number of template or function argument parameters.
Default value is 11, but can be overridden at compile-time.


    #ifndef NICTA_MAX_ARITY
    # define NICTA_MAX_ARITY 11
    #endif
